Bulk Carrier Interview Trainer — Native Android APK Project

Что внутри:
- Нативное Android-приложение, не HTML.
- 150 вопросов для Chief Officer / Master Bulk Carrier.
- Вопрос на английском.
- Ответ на английском.
- Объяснение на русском.
- Continuous playback: question -> answer -> Russian explanation -> next question.
- Foreground audio service with notification controls.
- Partial WakeLock to help playback continue when screen is off.
- Play all / Pause / Resume / Stop.
- Play from selected question.
- Filter by section and search.

Важно:
- Android TextToSpeech не поддерживает идеальную паузу внутри фразы.
  Pause останавливает текущую фразу, Resume повторяет текущий блок и продолжает дальше.
- Для сборки APK откройте папку проекта в Android Studio и нажмите:
  Build -> Build Bundle(s) / APK(s) -> Build APK(s)

Package:
com.bulkcarrier.interviewtrainer

Recommended:
- Android Studio Ladybug или новее.
- Compile SDK 35.
- Min SDK 26.

Если Android Studio спросит синхронизацию Gradle, нажмите Sync.
